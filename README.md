架构:

控制面:
vcpe由路由平面和转发平面组成, 控制器通过bgp向vrr下发配置和路由并转发至vpe, vpe通过bgp下发路由和配置到vcpe/cpe
在快速实现中, 控制器通过netconf/restful api控制vcpe与vpe建立bgp邻居关系,并下发其他配置

数据面:
vcpe与vpe/vcpe建立vxlan over ipsec隧道, 数据通过wan口/ipsec隧道/vxlan over ipsec隧道转发

功能:
注册/自动上线
配置管理
接口管理
ipsec
nat
vxlan
路由
转发
acl
dpi
安全
